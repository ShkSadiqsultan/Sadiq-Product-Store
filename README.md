# Sadiq Product Store

Welcome to the Sadiq Product Store! This is a simple Angular application that allows users to browse products, add new products (for logged-in users), manage a shopping cart, and view a dashboard of users.

## Live Demo

You can view the live application at the following link:

[Sadiq Product Store](https://sadiq-product-store.vercel.app)

## Features

- **Home**: Displays all products fetched from the Fake Store API.
- **Add Product**: Allows logged-in users to add new products.
- **Cart**: Users can add items to their shopping cart and view the total number of items.
- **Dashboard**: Displays a list of users (only accessible to logged-in users).

## Technologies Used

- Angular (version 13.3.0)
- Bootstrap for styling
- Font Awesome for icons
- Reactive Forms for form handling
- Fake Store API for product data

## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```
2. Run npm install
3. Serve the app `ng serve`
